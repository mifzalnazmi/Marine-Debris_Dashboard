# Marine-Debris_Dashboard
Dashboard interaktif ini digunakan untuk menganalisis sebaran marine debris di jawa barat berdasarkan data survei 2013–2018. Fitur utamanya meliputi peta densitas sampah, KPI, tren temporal, hotspot pencemaran, analisis korelasi, serta wind rose &amp; wave rose untuk mendukung pengambilan keputusan mitigasi &amp; pengelolaan sampah laut berbasis data.
